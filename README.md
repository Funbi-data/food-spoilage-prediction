# Food Spoilage Prediction

## Overview
This project analyzes environmental and sensor data to predict food spoilage using machine learning.

## Key Features
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Logistic Regression and Random Forest models
- Feature importance analysis

## Results
- Logistic Regression Accuracy: ~96%
- Random Forest Accuracy: ~98%

## Key Insights
- Temperature and humidity significantly influence spoilage
- Gas emissions are strong indicators of decomposition
- Nonlinear models improve predictive performance

## Tools Used
- Python
- Pandas, Seaborn, Matplotlib
- Scikit-learn
